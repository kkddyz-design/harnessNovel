【生成要求】
你的任务是为当前批次生成一份完整的情节摘要。须承接前序批次的情节发展，同时参考对应的参考批次摘要的节奏和张力曲线。

输出格式：
将批次内容按故事片段组织。一个故事片段覆盖2-5章，构成一个完整的起承转合小弧。不要逐章拆分，按情节的自然段落组织。

每个片段以【片段N：第X-Y章】开头（N从1开始），包含以下字段（每个字段都必须完整填写）：

1. 核心事件：本片段发生了什么。写清起因、经过、结果。包含具体的角色名、势力名、地名等专有名词。体现主角的行动链路：计划→遭遇什么→结果如何→调整后如何应对。

2. 矛盾与情绪：本片段的矛盾方向（对外冲突或对内修炼/心理）。情绪曲线，重点标注压抑点和爆发点。标注片段结束时主角的情绪基调。

3. 角色行动：主要角色的关键行动序列，用箭头连接。重点记录谁做了什么、为什么做、导致了什么后果。

4. 主角状态：本片段结束时主角的身份、实力、处境、认知状态。与片段开始时相比有什么变化。

5. 伏笔与线索：新埋伏笔及埋设方式、回收的前文伏笔及回收方式、信息差（谁掌握关键信息而谁不知情）。

6. 承接关系：这个片段与上一个片段的衔接方式（因果承接、时间跳跃、场景切换等）。本片段末尾为后续设置的悬念或铺垫。

所有片段之后，按以下板块输出整批汇总信息。每个板块必须完整，不得跳过。无相关内容的板块写"无"。

# 力量体系

# 新出场角色
本批内首次出场的角色。每个角色包含：姓名 | 身份 | 能力特长 | 与主角关系 | 核心动机

# 关键角色状态
本批结束时各关键角色的处境和变化。

# 主角成长
本批内主角的变化：实力突破、地位变化、资源获取、认知提升。标注触发点和章节。

# 逐章节奏
每章单独标注：节奏（快/中/慢） | 章末钩子类型（悬念/危机/反转/新谜团/信息揭露） | 张力等级（1-5）。汇总为本批节奏曲线（如"快-快-慢-中-快"），标注张力峰值章节。

# 跨章因果链
第X章的事件 → 第Y章的结果。标注因果传递链条，帮助理解事件之间的驱动关系。

【输出要求】
使用纯文本输出，禁止使用 Markdown 格式符号（如 *、**、-、>、| 等）。标题使用 # 标记。段落之间用空行分隔。
