【生成要求】
为本章写一份完整的章纲，包含以下 8 个要素。每个要素使用 # 标题作为独立段落，段落之间空行：

# 主线与状态
本章开头，主角（或视角人物）处于什么处境，想要做什么或面临什么问题。

# 事件流
按发生顺序叙述核心事件。每个事件写清：发生了什么、人物怎么反应、导致了什么结果。标注转折点（计划被打断、意外发生、真相揭露等）。
注意提取角色在对话或行动中使用的具体策略、话术、心理战术，以及这些策略如何逐步改变对方的心理状态。

# 关键细节
推动情节发展的具体内容，包括：
关键对话：必须写出具体内容（谁说了什么），不要概括为"说了xxx"。
关键行动：角色的具体行为及其直接后果。
关键发现：角色获得的重要信息、物品、线索。
关键决策：角色做出的重要选择及其理由。
内心活动：角色的内心独白、潜意识反应。
隐藏动机：角色抱怨、嫉妒、敌意等负面情绪的真实来源。

# 情绪走向
视角人物的情绪变化脉络。标注关键情绪节点（如危机感、疑惑、震惊、释然等），以及情绪转折的具体触发原因。

# 伏笔与信息差
本章新埋的伏笔、回收的前文伏笔、谁知道了什么而谁还不知道。

# 章末钩子
本章结尾停在什么位置。读者此刻最想知道什么、最担心什么。标注钩子类型（悬念/危机/反转/新谜题）。

# 本章角色
列出本章出场的所有角色：姓名、本章中的身份、与主角关系。

# 本章世界观元素
本章涉及的世界观相关内容：力量体系、组织体制、地名、技能/法术/功法使用等。

每章章纲 800-1500 字。

【输出格式】
使用纯文本输出，禁止使用 Markdown 格式符号（如 *、**、-、>、| 等）。

格式要求：
每一章的章纲必须以【第X章 章纲】开头。
8 个要素段落每个都以 # 标题 开头，段落之间空一行。

正确格式示例：

【第1章 章纲】
# 主线与状态
主角方宇穿越成青云宗杂役，干满三年可入外门修仙。

# 事件流
正当他得意于完成任务时，陈师兄一身伤痕到访，告诉他上了宗门二长老给邪门输送耗材的名单。不等他细问，陈师兄飞身离开。他正犹豫跑还是留，伙房外一阵喧闹，得知宗门中品灵宝被偷，贼人正是陈师兄。执法弟子问他是否见过陈师兄，他立时否定。回到房间正准备收拾包袱，一打开房门，房中竟然散发出一阵诡异青光。

# 关键细节
陈师兄说"二长老每个月从杂役里挑三个人，送到后山那个地方去，去了的人再没回来过"。方宇决定先假装不知情再找机会离开。

# 情绪走向
得意→危机感→疑惑→震惊

# 伏笔与信息差
陈师兄提到的"后山那个地方"尚未解释。方宇知道秘闻但执法弟子不知道。

# 章末钩子
房中青光是何物？耗材名单是真是假？（反转/悬念）

# 本章角色
方宇（主角，青云宗杂役）、陈师兄（青云宗外门弟子，告知方宇秘闻后逃离）、执法弟子（青云宗执法堂弟子）

# 本章世界观元素
青云宗（组织）、外门/内门/杂役（宗门等级）、中品灵宝（法宝等级）
