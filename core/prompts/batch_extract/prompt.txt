【提取要求】

按故事片段组织，每个片段必须以【片段N：第X-Y章】开头（N从1开始，X和Y是章节号，必须使用中文方括号【】）。

每个片段包含以下字段（所有字段都必须填写，不得省略）：

1. 核心事件：本片段发生了什么。写清起因、经过、结果。包含具体的角色名、势力名、地名、法宝名、功法名等专有名词，不要概括化。体现主角的行动链路：计划→遭遇什么→结果如何→调整后如何应对。

2. 矛盾与情绪：标注本片段的矛盾方向（对外冲突或对内修炼/心理）和情绪曲线。情绪曲线重点标注压抑点和爆发点。如果有多次压抑后爆发的节奏（如被质疑→再被质疑→彻底爆发），标注压抑累积的过程。标注片段结束时主角/读者的情绪基调。

3. 角色行动：主要角色的关键行动序列，用箭头连接。重点记录谁做了什么、为什么做、导致了什么后果。

4. 主角状态：本片段结束时主角的身份、实力、处境、认知状态。与片段开始时相比有什么变化。

5. 伏笔与线索：新埋的伏笔及埋设方式、回收的伏笔及回收方式、信息差（谁掌握关键信息而谁不知情）。

6. 承接关系：这个片段与上一个片段的衔接方式（因果承接、时间跳跃、场景切换、视角转换等）。本片段末尾为后续设置的悬念或铺垫。


在所有片段之后，按以下板块输出整批汇总信息。每个板块必须完整，不得跳过。无相关内容的板块写"无"。

# 力量体系

# 新出场角色

# 已有角色更新
本批中已有角色的新信息：新能力觉醒、技能成长（已有技能的新表现、新阶段、新用法）、身份揭露、关系变化、情感进展、信息获取等。按角色名分条，标注变化发生的章节。

# 地图场景

# 主角成长
