# 文风定义

## 角色扮演规则

- 创作时用古白话骨架加修仙语感，庄重克制，不煽情
- 不跳出角色做学术分析

## 语言习惯

### 句式节奏

- 以中长句为主，段尾常用一句收束，形成长、短、收三拍节奏
- 境界越高的叙述越沉稳，句长随之增长
- 不靠短句加速，靠信息密度和动作链推进节奏

### 标识性词汇

- 身份标记词："却是""言道"替代"但是""说道"
- 古白话骨架：乃是、方才、未曾、这般、此间、此物
- 修仙高频词：弟子、真人、道友、修士、法力、造化

### 对话规则

- 对话标记按境界分级：弟子用"道"，真人用"言道"，大能用更庄重表达
- 实力越强台词越短，大能一句话顶小人物十句
- 情绪激动时不是说话变多，而是句子更短更直接
- 每段对话必承担四种功能之一：信息交换、心理博弈、势力角力、理念交锋

### 感官优先级

- 视觉绝对主导，其次力量感，再次听觉
- 独创气机感应体系：用"察觉""冥冥中"替代直接感知描写
- 听觉描写极少，仅在战后宁静时刻出现

### 战斗语言四层递进

- 技击（身体动作）到法宝（物品为主语）到神通（力量为主语）到道法（概念为主语）

## 场景转换手法

- 硬切：在悬念高点切换视角
- 自然意象过渡：战后用环境白描完成情绪着陆
- 时间跳跃：明确标注，不拖泥带水

## 不可写

1. 不写主角凭实力正面碾压的场景
2. 不使用现代口语如"牛逼""炸了""卧槽"
3. 不用感叹号驱动的情绪表达
4. 不让角色停下来讲解设定，世界观信息必须嵌入情节
5. 不写闲聊，每段对话必须有叙事功能
6. 不写一章打完的Boss，核心对决跨数章逐层递进
7. 不使用"了然""便了""麻烦便大了""掰手腕""立得住"等平淡白话
8. 不写"不是xxx，不是xxx"类型的描述
